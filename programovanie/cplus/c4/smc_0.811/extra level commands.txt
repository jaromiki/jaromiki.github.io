You can add special level commands in the .txt file of your level.

Open your level file with an text editor and add for :



- ### Music :

Music land_1.mid 0 45
		|    |  |
		1    2  3

1 : Music filename
2 : Force it to play when entering the level ?
3 : Volume 0 - 128




- ### Modified Camera :

Mod_Camera 0 0 0
           | | |
	     1 2 3

1 : up
2 : left
3 : right




- ### Modified background Color :

Background_Color 0 0 0
		     | | |
		     1 2 3

1 : Red
2 : Green
3 : Blue
