Game keys last modified 12.12.2003

##### Moving #######

- Keyboard :

Moving left = Left
Moving right = Right

- Joypad :

Moving left = Joy Left
Moving right = Joy Right


##### Actions #######

- Keyboard :

Jump = Up
Duck = Down
Shoot = Space

- Joypad :

Jump = Joy Button 0
Duck = Joy Down
Shoot = Joy Button 1

##### Leveleditor ####

- Keyboard :

Open/Close Leveleditor = F8
Select an Object to place = F1
Save Current changes = SRTG + S

Copy an Object under the Mouse = STRG + C
Paste the copied Object to the Mouse Position = STRG + V

Modify Object to passive/massive/halfpassive = m

- Precision Positioning :

Left = Numpad 4
Right = Numpad 6
Up = Numpad 8
Down = Numpad 2

- Fast Map Scrolling

Left = n
Right = p

- Mouse :

Move Objects : Mouse Buton 1 + move the mouse
Delete Objects : Mouse Button 2


##### Misc ######

- Keyboard :

Back to Main menu = ESC
Make an Screenshot = F5
Enable/Disable Sounds = F10
Enable/Disable Music = F11

- Joypad :

Back to Main menu = Joy Button 2
