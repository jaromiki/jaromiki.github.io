Podprogramy

Každý C-program sa skladá z niekoľkých podprogramov (každý program obsahuje funkciu main() ), preto je dôležité naučiť sa ich správne používať. Podprogramy musíme zadefinovať (väčšinou nad funkciou main() ), co obsahuje
hlavičku - typ nazov (parameter) .
telo - blok - v strapatých zátvorkách postupnosť príkazov, môžu tu byť definované aj premenné platné len v tomto bloku. Podprogramy možno rozdeliť na:
  • procedury - sada príkazov dostane nejake pomenovanie (vznikne nový príkaz s týmto menom). Procedury nemajú návratovú hodnoty, ich úlohou je niečo vykonať napr. na obrazovke, prípadne nastaviť/zmeniť viaceré globálne premenné.
    (takýto príkaz potom voláme zapísaním jeho názvu, kde v zátvorke môžu byť parametre
    napistrikratA(); //program odskočí do popisu podprogramu, kde nájde sadu príkazov, ktoré vykoná a vráti sa späť kde bol volaný.)
    Vytvoriť "čistú" procedúru možno určením jej typu ako void.
  • funkcie - majú návratovú hodnotu, ich úlohou je teda túto hodnotu vypočítať a zapísať do výrazu, kde je funkcia volaná.
    (napr. y=sin(x)+1; //výraz na pravej strane obsahuje volanie funkcie sin(), program si odskočí do podprogramu kde z parametra x postupnosťou príkazov vypočíta hodnotu sínusu a táto vypočítaná hodnota nahradí text sin(x) vo výraze.)
    Aj funkciu teda možno chápať ako výraz a teda aj možno vytvoriť výrazový príkaz pripísaním bodkočiarky
    (napr. sin(x); //co v pripade fun. sinus moc vyznam nema)
Každý nami vytvorený podprogram možno volať/použiť z funkcie main() a tiež v každom podprograme môže byť zavolaný iný podprogram. V jazyku C platí, že pri čítaní programu zhora dole, ak narazíme na volanie podprogramu [napr. aa()] musí byť daný podprogram [v tomto prípade aa()] už zadefinovaný, inak kompilátor zahlási chybu lebo danému pojmu nerozumie. Často stačí určiť správne poradie definícií podprogramov aby tento problémik nenastal, no pri väčšom počte podprogramov by to mohlo byť dosť obtiažne. Je tu riešenie v tzv. vymenovaní hlavičiek všetkých podprogramov, ktoré v programe používame. Toto voláme deklarácia podprogramu, v deklarácii na poradí hlavičiek nezáleží. Deklaráciu umiestnime nad popis/definíciu podprogramov a program, keď objaví volanie nejakého podprogramu, tak nezačne robiť paniku, lebo vie, že ide o vlastný podprogram a bude veriť, že jeho celé telo sa dozvie pri ďaľšom čítaní programu.
Deklarácia
V C-čku je rozdiel medzi deklaráciou (len sa určí typ funkcie a typ parametrov) a definíciou (kde je už aj telo). Zaujímavou skutočnosťou je, že viaceré funkcie môžu mať rovnaké meno - v tomto prípade sa musia líšiť v typoch alebo počte parametrov
(Viac pozri prekrývanie funkcií)
Syntax:
typ názov(param);
- typ - všetky jednoduché typy, štruktúry, smerník. Ak je typ vynechaný je chápaný ako int. Na procedúry použijeme typ void.
- (param) - v okrúhlych zátvorkách po jednom zadefinované parametre, no stačí zoznam typov formálnych parametrov
           - tiež je dovolený typ struc, pole
Pozn:
Deklarácia je teda len hlavička funkcie a musí byť ukončená bodkočiarkou!
Príklad:
int sucet(int, int);
void vypis();
float maxim(float a[], int n);
Definícia
Tu je popis názvu podprogramu, parametrov a tiež postupnosť príkazov, ktoré sa majú vykonať pri volaní.
Syntax:
typ názov(deklar_param)
blok

- typ - všetky jednoduché typy, štruktúry, smerník. Ak je typ vynechaný je chápaný ako int. Na procedúry použijeme typ void.
- deklar_param - parametre sú pojednom zadeklarované (oddelené čiarkou).
Niektoré parametre slúžia nato aby len priniesli do podprogramu hodnotu - takéto parametre voláme vstupné alebo parametre prenášané hodnotou. Niektoré parametre slúžia na prenos údajov do podprogramu a zároveň na prenos údajov z podprogramu - takéto voláme vstupno-výstupné alebo parametre prenášané adresou.
Štandartne sú parametre prenášané hodnotou (sú len vstupné), ak je parameter pole, tak je prenesený adresou.
- blok - V bloku je postupnosť príkazov. Ak typ funkcie je rôzny od void tak v každej vetve podprogramu sa musí nachádzať príkaz return s návratovou hodnotou. Príkaz return ukončí vykonávanie podprogramu a výsledok výrazu uvedeného za slovom return je návratovou hodnotou.
Ak je typ podprogramu void, tak príkaz return môže byť vynechaný a podprogram je ukončený ak nájde poslednú strapatú ukončovaciu zátvorku.
Ak je typ podprogramu void, tak príkaz return bez návratovej hodnoty môžme použiť na ukončenie podprogramu.
Pozn:
Hlavička funkcie a nesmie byť ukončená bodkočiarkou!

Príklad: 

int sucet(int a, int b)  //oba parametre sú vstupné, vzniknú pri zavolaní tohoto podprogramu
                         //hodnotu dostanu podla uvedenych parametroch pri volani
                         //po skončeni podprogramu tieto premenne zaniknu
                         //ich pripadna zmena v podprograme nema vplyv na ziadnu globalnu premennu
{ return a+b;}
Volanie:
Všetky funkcie možno volať aj ako samostatný výrazový príkaz.
Nevoidové funkcie možno používať vo výrazoch
Skutočné a formálne parametre musia byť kompatibilné na priradenie.
Príklad: 
int main()
{ int b=1,c=2;
  cout << sucet(b,c) << endl;               // vypise 3
  cout << "b=" << b <<"   c=" << c << endl; // volanie podprogramu nema vplyv na premenne b,c
}
Príkaz return
Návratová hodnota je určená výrazom za kľúčovým slovom return.
Syntax:
return hodnota
Sémantika:
ukončí podprogram a hodnotu získa identifikátor funkcie.
voidovú funkciu možno ukončiť príkazom return bez hodnoty.
Funkciu, ktorá nemá návratovú hodnotu možno ukončiť príkazom:
return ; // teda bez zátvoriek a bez zadanej hodnoty
alebo možno aj celý príkaz return vynechať - vykonávanie funkcie je ukončené nájdením ukončovacej zátvorky }

Volanie takejto funkcie je volanie príkazu - pozri nasledovný príklad.

Procedúra - 
void vypis() //aj ked funkcia nema param. musí mať zátvorky

Takto zadeklarovaný podprogram možno zavolať ako samostatný príkaz:

Funkcie so vstupnými param.

#include <iostream>
using namespace std;
int maxint(int a,int b)
{ return (a>b?a:b); }
// funkcia vypočíta maximum z dvoch celých čísel
void main()
{ cout << "max. z cisel 15 20 je:" << maxint(15,20);
    cin.get();}

skutočné parametre môžu byť aj výrazy , ak je skutočný parameter premenná, tak jej hodnota sa zavolaním funkcie nemôže zmeniť.
Parametre
  • predávané hodnotou - klasické vstupné parametre, pri volaní sa vytvoria plnohodnotné premenné s inicializáciou poč. hodnoty.
    Napr.
    int sucet(int a,int b,int c)
    { return a+b+c; }
  • predávané menom - vstupno-výstupné parametre, uvedené znakom &. Tu žiadna nová premenná nevzniká, len premenná zapísaná vo volaní dostala ďalšie meno.
    Napr.
    int vymena(int &a,int &b)
    { int c; c=a;a=b;b=c; }
  • predávané adresou - vstupno-výstupné parametre, uvedené znakom *. Prenáša sa teda hodnota=adrese dát.
    Napr.
    int vymena(int *a,int *b)
    { int c; c=*a;*a=*b;*b=c; }
  • parametre s default hodnotami
    Niekoľko posledných parametrov môže mať predpísané "zástupné" hodnoty, ktoré sa použijú ak parameter pri volaní nie je zadaný

    int sucet(int a,int b=0,int c=0,int d=0)
    { return a+b+c+d; }
    //takýto podprogram možno volať s jedným až štyrmi celočíselnými parametrami.
parametre s default hodnotami
Funkcia môže predpísať parametrom default hodnoty pre niektoré parametre, ktoré sa použijú ak ich neuvedieme pri volaní

#include <iostream>
#include <climits> //tu su zapisane konstanty INT_MIN
using namespace std;
int maxint(int a,int b=INT_MIN,int c=INT_MIN,int d=INT_MIN)
{ a = (a>b?a:b); b = (c>d?c:d);
return (a>b?a:b); }
// funkcia vypočíta maximum z dvoch až 4 celých čísel
void main()
{ cout << "max. z cisel 15 20 je:" << maxint(15,20);
  cout << "max. z cisel 15 20 -5 je:" << maxint(15,20,-5);
  cout << "max. z cisel -100 15 20 -5 je:" << maxint(-100,15,20,-5);
  cout << "max. z cisel 15 je:" << maxint(15);
  cout << "max. z cisel je:" << maxint(); // zahlasi chybu
}
}

Funkcie s neurčeným počtom parametrov
Funkciu možno napísať tak, že neurčíme počet parametrov. Môžme určiť niekoľko pevných parametrov (obyčajne tu uvedieme parameter v ktorom je zapísaný počet ďalších param.)

#include <stdio.h>
#include <stdarg.h>
int sucet(int poc ...)
va_list zoz; // tu budu nejak zapisane "utajene" arg
int s=0; // v s bude sucet cisel,

va_start(zoz,poc); va_start inicialiyuje zoznam par, druhy param mozme zadat :hocico, ale dava sa pevny parameter
for (int i=0;i<poc;i++)
  s += va_arg(zoz, int); // vyberie "utajeny" argument a nastavim kurzor na dalsi
va_end(zoz); // ukonci spracovanie parametrov
{ return (s); }
// funkcia vypočíta maximum z dvoch až 4 celých čísel
void main()
{printf("sucet z cisel %i, %i, %i je: %i",1,2,3,sucet(3,1,2,3));
}

Funkcie používame ak chceme aby podprogram vypočítal jednu hodnotu a tú nám vrátil na miesto kde bol zavolaný.
(najčastejšie niekam do výrazu)
Napíšeme podprogram/funkciu median3() na nájdenie mediánu z troch celých čísel (stredné číslo) 
 #include <iostream>
 using namespace std;
 int median3(int x,int y,int z)
 { if (x>y) 
   {   if (y>z) return y; 
	if (z>x) return x;
	else     return z;
   } 
   else
   {   if (x>z) return x; 
	if (z>y) return y;
	else     return z;
   } 

}
 int main()
 { int a=random(10),b=random(10),c=random(10) ,d=1;
   cout <<a<<"  "<<b<<"  "<<c<<"  median= "<<median3(a,b,c)<<endl;
   while(cout <<"pokracovat 1/0 :", cin >>d, d)
   {  a=random(10),b=random(10),c=random(10);
      cout <<a<<"  "<<b<<"  "<<c<<"  median= "<<median3(a,b,c)<<endl;
   }
   return 0;
 }

Funkcia s parametrami typu pole
Použitie poľa ako parametra je spojené s niektorými odlišnosťami
  • Pri definovaní parametra typu pole neuvádzame počet prvkov
    {napr int max_pola(int pole[]) }
  • podprogram nevie zistiť koľko prvkov pole=parameter vlastne má a je zvykom pri použití poľa ako parametra vytvoriť ešte jeden int parameter cez ktorý sa podprogramu oznámi, koľko prvkov to pole vlastne má.
    {napr. int max_pola(int pole[],int n) }
  • Pri volaní podprogramu s parametrom typu pole musí byť skutočným parametrom opäť pole a malo by byť rovnakého typu (jednoduchý parameter môže byť pri volaní zastúpený aj výrazom - nemusí tam byť premenná)
  • Ak v podprograme zmeníme prvok poľa, zmení sa zároveň aj pole, ktoré je uvedené ako parameter pri volaní
    ( zmena jednoduchého parametra v podprograme nemá vplyv na parametre uvedené pri volani podprogramu)

Napíšeme podprogram/funkciu max_pola() na nájdenie maximu celočíselného poľa ktoré bude ako parameter podprogramu 
 #include <iostream>
 using namespace std; 
int a[100] , p=20;
//spravny podprogram nepouziva globalne premenne, ak premennu potrebuje tak si ju vytvori
int max_pola(int pole[],int n)
{  int m=pole[0];
   for (int i=1;i<n; i++) if (pole[i]>m) m=pole[i];
   n=0; //to len aby sme vo volaní videli, že premenná p sa nezmení
   pole[0]=1; //to len aby sme videli, ze pole pouzite pri volani sa zmeni (teda prvok na adrese 0) 	
   return m; 
}

int main()
{  for (int i=0; i<p; i++) cout << (a[i]=10+random(90)) <<"  ";   //naplnenie+vypis
   cout <<endl<<"maximum: "<< max_pola(a, p)<<endl;
   cout <<"premenna p= "<<p<<endl;
   cout <<"a[0]= "<<a[0]<<endl;
   system ("pause");
   return 0;
 }

Procedúra s parametrami typu pole
Procedúra nemá návratovú hodnotu, úlohou podprogramu bude teda nejak zmeniť pole, ktoré bude poslané do podprogramu cez parameter.
Naprogramujeme podprogram vpravo(), ktorý posunie všetky prvky na číslo izby o jedno číslo väčšie a posledný prvok presunie do izby číslo nula.
Riešenie: 
 #include <iostream>
 using namespace std;
 int a[100],  n=30;
 void vpravo(int pole[],int n)
 { int p=pole[n-1];
   for (int i=n-1; i>0; i--) pole[i]=pole[i-1]; 
   pole[0]=p; 
 }
 int main()
 { for (int i=0; i<n; i++) cout << (a[i]=10+random(90)) <<"  ";   //naplnenie+vypis
   vpravo(a,n);
   for (int i=0; i<n; i++) cout << a[i] <<"  ";   //vypis
   system("pause");
   return 0;
 }

Podprogramy s viacerými výsledkami
Môžeme písať aj podprogram s viacerými výsledkami. Keďže výsledkov je viac musíme výsledky zapísať do niektorých parametrov. Parametre, ktoré majú aj "vyniesť" výsledok musia byť "špeciálne" zadefinované (sú označené tak, že pred meno majú pridaný znak &). Ukážem to na príklade podprogramu
utried3(), ktorá utriedi 3 premenné, ktoré tvoria vstupné pamametre.
void utried3(int &a, int &b, int &c)
{ int d;
  if (a>b) {d=a; a=b; b=d;}
  if (b>c) {d=b; b=c; c=d;}
  if (a>b) {d=a; a=b; b=d;}
}
int main()
{ int a=7,aa=5,aaa=6;
  cout <<a<<' '<<aa<<' '<<aaa<<endl; // 7 5 6
  utried3(a,aa,aaa);
  cout <<a<<' '<<aa<<' '<<aaa<<endl; // 5 6 7
  system("pause");
  return 0;
}

Parametre a,b,c sú aj vstupné (prinesú hodnoty do podprogramu), aj výstupné (odovzdajú výsledok do premenných do miesta, kde bol volaný). Takéto parametre voláme vstupno-výstupné. Niekedy je vhodné použiť na vstup vstupné parametre a na výsledky parametre výstupné.
Napíšeme podprogram na zistenia min a max z troch premenných minmax()
void minmax(int a, int b, int c,int &min, int &max)
{ min = a<b?a:b;
  if (min>c) min=c;
  max = a>b?a:b;
  if (max<c) max=c;
}
int main()
{ int a=7,aa=5,aaa=6,b,c;
minmax(a,aa,aaa,b,c);
  cout <<"min=" <<b<<endl; // 5
  <<cout "max=" <<c<<endl; // 7
  system("pause");
  return 0;
}
