Príkaz if
Syntax
if (celociseny_vyraz) prikaz1
if (celociseny_vyraz) prikaz1 else prikaz2
Sémantika
ak je celociseny_vyraz rôzny od nuly urobí sa príkaz1, ak je =0 urobí sa príkaz2
Relačné operátory: >, >=, <, <=, ==, != Logické operátory na tvorbu zložitých logických výrazov (samozrejme, že logické výrazy sú v C celočíselné):
&&, ||
Pozn.
Aj pred else musí byť bodkočiarka
Príklady:
  1. Načítaj rok a zisti čí je rok priestupný
  2. Načítaj celé číslo a vypíš najbližšie celé delitelné 3
  3. Načítaj tri celé čísla a vypíš ich utriedené
Príkaz switch - obdoba pascalského case
Syntax
switch (celociseny_vyraz)
{ case konst_vyraz1 : prikazy1
case konst_vyraz2 : prikazy2
...
default: prikazy
}
Vetva default je nepovinná, oproti pascalu v každej vetve môže byť aj viac príkazov.
Sémantika
  • Vyhodnotí sa cečíselný výraz
  • hľadá konšt. výraz požadovanej hodnoty
  • ak nájde pokračuje od príkazov v danej vetve (vzkoná aj príkazy vo vetvách pod ňou)
  • ak nenájde pokračuje od vetvy default
  • ak vetva defaut nie je nevykoná sa nič
Pozn.:
Ak chceme podobný príkaz ako pascalský case musíme na konci každej vetvy použiť príkaz
break;
ktorý ukončí vykonávanie zloženého príkazu
Príklad
switch.cpp
  #include <stdio.h>         
  void vystup(int n)            
  {                         
    switch n
	{
	 case 1: puts("vyborny");break;
	 case 2: puts("chvalitebny");break;
	 case 3: puts("dobry");break;
	 case 4: puts("dostatocny");break;
	 case 5: puts("nedostatocny");break;
	 default: puts("-----------");break;
	}
  }
Úlohy:
Načítaj dátum - den, mesiac, rok a zisti, či je korektný.
Načítaj 3 reálne čísla a vypíš ich utriedené
Načítaj znak a vypíš či ide o spoluhlásku, samohlásku, číslicu, iný (špec.) znak
