Cykly používame na opakované vykonávanie nejakého príkazu. Jazyk C podobne ako pascal pozná 3 druhy cyklov.
Cyklus for
Syntax:
for (start_vyraz ; podm ; iter_vyraz ) prikaz
start_vyraz - ľubovoľný výraz, ktorý sa vykoná pred cyklom (iba raz
podm - podmienka zotrvania cyklu - celočíselný výraz
iter_vyraz - ľubovoľný výraz, ktorý sa vykoná vždy po vykonaní príkazu
prikaz - ľubovoľný príkaz, ktorý sa bude opakovane vykonávať
Sémantika:
  1. vyhodnotí sa startovací výraz
  2. vyhodnotí sa podmienka
  3. ak podm != 0 tak sa urobí príkaz , potom iteračný výraz a potom späť na 2.
  4. ak podm = 0 cyklus končí
Poznámky:
... všetky údaje v zátvorke sú nepovinné, t.j. možno ich vynechať, dve bodkočiarky sú nutné
     for (;;) vypocet; // je nekonečný cyklus
... ak chýba podmienka je implicitne 1 teda TRUE
... v C++ je možné v start_vyraz premennú zadeklarovať int s=0;for (int s=0,i=1;i<11;i++) s+=i;
     tento cyklus spočíta čísla 1 - 10
     premenná i je viditelná jen v cykle, po skončení cyklu neexistuje
... je zvykom za for písať medzeru (pre odlíšenie od podprogramu, kde medzeru nedávame)
... ak chceme vynechať príkaz musíme dať prázdny príkaz
... ide o veľmi mohutný príkaz, ktorý obsahuje väčšie množstvo kódu, preto ak sa dá, nahrádzame ho cyklami while, resp. do while

Cyklus while - cyklus s podmienkou na začiatku
Syntax:
while (podm ) prikaz
podm ... ľubovoľný celočíselný výraz
Sémantika:
pokiaľ je podm != 0 vykonáva sa príkaz
Pozn.
za slovom while je zvykom písať medzeru aby sa zvýraznilo, že nejde o funkciu
Príklad
Napíšeme program, ktorý načítava zo vstupu znaky až pokiaľ nenačíta koniec súboru a vypisuje znaky na štandartný výstup
prog.cpp

  #include <stdio.h>         //pripojenie kniznice
  void main(void)            //hlavicka "hlavneho podprogramu"
  { char z;                         //zaciatok bloku
    while ((z=getchar())!=EOF) putchar(z); 
      //preco musi byt priradenie v zatvorkach? 
  }

Zdá sa, že tento program nemá žiadne využitie, ale chcem ukázať ako takýto jednoduchý program môže byť silným nástrojom. Vyrobme si jeho exe, resp. a.out v UNIXe.
príkazom: prog <prog.cpp dosiahneme výpis súboru prog.cpp na obrazovku (vlastne obdoba príkazu type, view)
príkazom: prog >subor.txt sme vytvorili obdobu DOSáckeho copy con subor.txt
príkazom: prog <prog.cpp >subor.txt okopírujeme súbor prog.cpp do subor.txt
Úlohy:
  1. Upravte posledný program pre UNIX, tak aby v prípade neúspechu zápisu znaku na štand. výstup vypíše na štand. chybový výstup hlásenie o chybe.
  2. na vstupe sú znaky ukončené enterom alebo koncom suboru načítajte znaky a vypíšte ich pod vstupné data
  3. na vstupe sú čísla ukončené enterom alebo koncom suboru načítajte čísla a vypíšte ich pod vstupné data a do ďalšieho riadku ich spočítajte
  4. na vstupe je číslo pri základe (-2) napr. 11-2=1*(-2)0+1*(-2)1 = 1 - 2 = -1 do ďalšieho riadku vypíš to číslo v 10-tkovej sústave a do ďalšieho v klasickej dvojkovej
  5. načítaj prirodzené a,b a vypočítaj NSD(a,b), NSN(a,b)


Cyklus do-while - cyklus s podmienkou na konci
Syntax:
do prikaz while (podm )
podm ... ľubovoľný celočíselný výraz
Sémantika:
vykoná sa príkaz
pokiaľ je podm != 0 vykonáva sa príkaz
Pozn.
príkaz sa vykoná aspoň raz
Úlohy:
  1. Na vstupe sú čísla ukončené 0, zisti ich súčet
  2. Vypočítaj a s presnosťou na 3 desatinné miesta
  3. Načítaj číslo n a zisti počet cifier v des. sústave


Príkaz break - ukončí cyklus
Syntax:
break;
Sémantika:
Používa sa v zložených príkazoch na ich ukončenie a pokračovanie až za }
Pokiaľ sa dá sa takejto konštrukcii cyklu s break vyhýbame
Príklad:
break.cpp

  #include <stdio.h>         //pripojenie kniznice
  void main(void)            //hlavicka "hlavneho podprogramu"
  { int cislo, sucet=0, pocet=0, naj=0;                         
    while ( scanf("%i", cislo) != EOF)
    {     if (cislo<0) break;
          sucet +=cislo; pocet++; 
          if (cislo > naj) naj=cislo;
    } 
  }


Príkaz continue - ukončí iteráciu cyklu
Syntax:
continue;
Sémantika:
Používa sa v zložených príkazoch v cykloch na ich ukončenie návrat na začiatok príkazu cyklu
Príklad:
continue.cpp

  #include <stdio.h>         //pripojenie kniznice
  void main(void)            //hlavicka "hlavneho podprogramu"
  { int cislo, sucet=0, pocet=0, naj=0;                         
    while ( scanf("%i", cislo) != EOF)
    {     if (cislo<0) continue;
          sucet +=cislo; pocet++; 
          if (cislo > naj) naj=cislo;
    } 
  }
