Delphi a MySQL

Autor: Jakub Dubec, korektura: Zdeněk Lehocký, 22. 04. 2010, 13 497× přečteno

Jednoduchý príklad na prácu s MySQL databázou v Delphi. Ukážeme si, ako sa pripojiť k vzdialenej MySQL databáze a ako z nej prečítame meno a priezvisko zamestnanca, jeho stručný životopis, zameranie, prax a jeho fotku.

Najskôr si v MySQL vytvorte novú databázu, ktorá sa bude volať test a bude obsahovať jednu tabuľku s názvom zamestnanci a nasledujúce atribúty (Obrázok 1):

  • Id (INT)
  • Meno (VARCHAR(45))
  • Priezvisko (VARCHAR(45))
  • Zameranie (VARCHAR(45))
  • Prax (INTEGER)
  • Zivotopis (TEXT)
  • Fotka (MediumBLOB)

Ďalej si vytvoríte užívateľa, napr. USER s heslom H3slo. Teraz vytvoríte v Delphi nový projekt s názvom zamestnanci (Obrázok 2): File → New → VCL Forms Application.

Na formulár (frmDB) umiestnite nasledujúce komponenty (Obrázok 3):

  • Image (fotka)
  • DBText (DBmeno)
  • DBText (DBpriezvisko)
  • DBText (DBzameranie)
  • DBText (DBprax)
  • DBMemo (DBzivotopis)
  • Button (btnNext)
  • Button (btnKoniec)
  • Label (LBzameranie)
  • Label (LBprax)

Pridajte do projektu nový DataModul (dmDB, File → New → Delphi Projects → Delphi Files → Data Module), na ktorý umiestnite nasledujúce komponenty (Obrázok 4):

  • DataSource (DataSource)
  • SQLConnection (SQLConnection)
  • ClientDataSet (DataSet)
  • SQLQuery (SQLQuery)
  • DataSetProvider (Provider)

Jednotlivé komponenty nastavte nasledovne:

  • DataSource:

    DataSet→DataSet

  • SQLConnection:

    ConnectionName→MySQLConnection
    Driver→MySQL
    LoginPrompt→False
    Params
    (Obrázok 5)    		→ HostName = localhost (adresa servera)
    Database = test (názov databáze)
    User_name = user (názov užívateľa)
    Password = H3slo (heslo užívateľa)
    Connected→True

  • DataSet:

    ProviderName→Provider
    Active→True

  • SQLQuery:

    DataSource→DataSource
    SQL→SELECT * FROM zamestnanci ORDER BY id ASC
    SQLConnection→SQLConnection
    Active→True

  • Provider:

    DataSet→SQLQuery

Ideme konečne programovať. Najprv pridajte do sekcií (Obrázok 6 a 7):

  • sekcia uses – Unit2, JPEG, DB – v Unit1
  • sekcia var – deklarovanie globálnej premennej aktualne

Teraz ošetrite udalosť TfrmDB.Create (POZOR! Fotky v databáze musia mať formát JPEG!):

var
jpeg: TJPEGImage;
bmp: TBitmap;
obrazok: string;
blob: TStream;
pocet: integer;
begin
if DirectoryExists ('temp')  = false then //ak neexistuje priecinok temp tak ho vytvori
CreateDir('temp');
pocet:=dmDB.SQLQuery.RecordCount; //zisti pocet prispevkov  v DB
if pocet < 0 then //ak sa v DB nachadza nenachazda ziadny prispevok vyhodi chybu
ShowMessage('Databaza neobsahuje ziadne prispevky') //chybova hlaska
else
  begin
    //ziskanie suboru z DB
   //precita fotku z DB a ulozi ju do premennej blob
    blob := dmDB.DataSet.CreateBlobStream(dmDB.DataSet.FieldByName('fotka'), bmRead); 
    //vytvori subor fotka.jpg, ktory bude v zlozke temp
    with TFileStream.Create('temp/fotka.jpg', fmCreate) do
      try
         CopyFrom(blob, blob.Size) //do suboru fotka.jpg sa pokusi ulozit fotku, ktoru nacital z DB
         finally
          free;
       end;
    //konvertovanie suboru
    //najprv musime skonvertovať subor fotka.jpg do BMP formatu pretoze TImage nedokaze zobrazit       JPEG subor
   obrazok:='temp/fotka.jpg'; //cesta k suboru fotka.jpg
   jpeg:= TJPEGImage.Create; //vytvori jpeg subor
   try
     jpeg.LoadFromFile(obrazok); //pokusi sa nacitat subor fotka.jpg
      bmp:= TBitmap.Create; //vytvori Bitmapu
      try
          bmp.Assign(jpeg); //prideli obrazok bitmape
          //ulozi bitmapu do priecinku temp, s nazvom fotka a da mu koncouku bmp
          bmp.SaveTofile(ChangeFileExt('temp/fotka', '.bmp' )); 
        finally
         bmp.free //uvolni bitmapu z pamati
       end
   finally
     jpeg.free //uvolni JPEG z pamati
    end;
    fotka.Picture.LoadFromFile('temp/fotka.bmp'); //zobrazi fotku v komponente fotka
   end;
  end;

Udalosť dtnKoniec.Click:

Application.Terminate;  //vypne aplikaciu

Teraz keď sa pokúsite spustiť aplikáciu, tak vám môže vyhodiť chybu (Obrázok 8).

Túto chybu spôsobilo to, že formulár bol vytvorený skôr ako DataModul. Túto chybu opravte tak že:

  1. pôjdete na Project → Options → Forms,
  2. presuniete dmDB nad frmDB (Obrázok 9).

Teraz keď spustíte aplikáciu, mala by fungovať.

Nakoniec nastavte komponenty DBMeno, DBPriezvisko, DBZameranie, DBPrax a DBZivotopis tak, aby zobrazovali obsah. Stačí nastaviť tieto vlastnosti:

DataSource→dmDB.DataSource
DataField podľa toho, čo ma zobrazovať; pri DBMeno to bude→meno

Teraz program funguje na 100 %. Ak program presuniete na iný počítač, nebude fungovať, pretože do koreňového priečinku aplikácie treba dať ešte tieto knižnice:

  • Libmysql.dll
  • Midas.dll
  • Dbxmys.dll
  • Dbxconnections.ini
  • Dbxdrivers.ini

Na stiahnutie:

  • Knižnice
  • Zdrojový kód celého programu
