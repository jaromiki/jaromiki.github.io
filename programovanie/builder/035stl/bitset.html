Zoznamy - list

Bitset je špeciálny kontajner, kde položky takéhoto objektu sú bity - teda hodnoty sú len 0,1 (TRUE, FALSE). Oproti bežnému poľu (ktoré má viac možností použitia) teda získame asi 8 násobne menší priestor, čo môže byť niekedy rozhodujúce. Celá práca s položkami bude spočívať v nastavovaní hodnoty a prípadne test hodnoty. Použitie je ako pri množine, vieme prvok vložiť, odstrániť, testovať.
pripnutie knižnice
Aby sme mohli používať obojstranné zoznamy musíme pripnúť príslušnú knižnicu
#include <bitset>
Po pripnutí môžme začať používať objekty popísané v tejto knižnici a teraz si základ popíšeme.
deklarácia
Je úplne podobné ako pri deklarácii poľa
  • bitset <n> bs1;      //vytvorí bitset dĺžky n bitov (vynulovaný)
  • bitset <8> bs1("00001111");      //vytvorí bitset dĺžky 8 bitov s predpísanou hodnotou
  •      pri inicializácii stringom je vhodné pripnúť aj knižnicu string (#include <string>)
členské funkcie
  • any() ... vráti TRUE ak sú všetky bity nastavené na 1
  • count() ... vráti počet bitov nastavených na 1
  • none() ... vráti TRUE ak sú všetky bity nastavené na 0
  • flip() ... zneguje všetky bity
    flip(n) ... zneguje n-ty bit
  • operator [n] ... sprístupní n-tý bit
  • reset() ... nastaví všetky bity na 0
    reset(n) ... nastaví n-ty bit na 0
  • set() ... nastaví všetky bity na 1
    set(n) ... nastaví n-ty bit na 1
    set(n,0) ... nastaví n-ty bit na 0
  • size() ... vráti počet bitov v bitsete
  • test(n) ... vráti TRUE/FALSE podľa hodnoty n-tého bitu
  • to_string,allocator >() ... vráti stringovú hodnotu
         (trochu komplikované ale treba to napísať presne takto)
    Project1
    // bitset::to_string
#include <iostream>
#include <string>
#include <bitset>
using namespace std;
int main ()
{ string mystring;
  bitset<4> mybits;     // mybits: 0000
  mybits.set();         // mybits: 1111
  mystring=mybits.to_string,allocator >();
  cout << "mystring: " << mystring << endl;
  return 0;
}
