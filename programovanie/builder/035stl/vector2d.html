Dvojrozmerné polia pomocou štruktúry vector

Ako pri poliach aj tu máme viacero možností ako vytvoriť dvojrozmernú tabuľku dát. Všeobecne môžme vytvoriť
  • pole vektorov - počet riadkov je pevný ak ide o statické pole
    vector<int> bar[10]; //vytvori pole 10-tich vektorov, všetky sú prázdne
    bar[0].resize(20); //nultej položke pola zmeníme počet prvkov na 20
    každému riadku môžme nastaviť iný počet stĺpcov
  • vektor polí - pole je možné použiť len dynamické (nebudeme používať - radšej namiesto poľa použijeme vector
    vector < int * > v(100, new int[10]); //100 riadkovy vector, v kazdom smernik na pole o 10-tich prvkoch
    každú položku vektora musíme dať vyrobiť cez new, lebo ide o dynamické pole
    máme premenlivý počet riadkov aj stĺpcov v každom riadku
    keď takýto vector ideme zmenšovať treba dynamické polia odstraňovaných riadkov najprv odstrániť
  • vector of vector
    vector < vector<int> > v(100, vector <int> (10,5)); //100 riadkovy vector, v kazdom vector o 10-tich prvkoch vsetky =5
Ďalej sa budem venovať už len prípadu vector of vector.
Použitie
Pri deklarovaní a zároveň vytváraní vectora môžme dať niekoľko prvkou hneď ak vyrobiť alebo nechať vector prázdny
  • Môžme použiť cez typedef vytvorenie nového typu: 
    // **** prázdny vektor 0x0 ********** //
typedef vector<int> Tvector ;
vector < Tvector > v;
Tvector pom(5,1);
v.push_back(pom); //vector v má už jeden riadok a ten má 5 prvkov - samé 1
v.resize(10); //  vector v má už 10 riadkov riadky 1-4 sú ale prázdne
//teraz riadkom 1-4 vytvorím stĺpce (i-ty riadok má i stĺpcov a ich počiatočná hodnota je i+1)
for (int i=1; i<5; i++) v[i].resize(i,i+1);

    // **** neprázdny vektor 100x0 ********** //
typedef vector<int> Tvector ;
vector < Tvector > v(100); //vytvoríme 100 riadkov, v každom je prázdny vektor
//** každému riadku teraz vytvorím 10 stĺpcov s počiatočnými hodnotami 5
for (int i=0; i<100; i++) v[i].resize(10,5);

    // **** neprázdny vektor 100x10 ********** //
typedef vector<int> Tvector ;
vector < Tvector > v(100, Tvector (10,1)); //vytvoríme 100 riadkov, vkaždom je vektor o 10-tich prvkoch
  • Bez použitia typedef 
    // **** prázdny vektor 0x0 ********** //
vector < vector<int> > v; 

    // **** neprázdny vektor 100x0 ********** //
vector < vector<int> > v(100); 

    // **** neprázdny vektor 100x10 ********** //
vector < vector<int> > v(100, vector <int> (10,5)); //100 riadkovy vector, v kazdom vector o 10-tich prvkoch vsetky =5
Príklady
  • Načítaj n a vytvor n-riadkovú tabuľku čísel pascalského trojuholníka. Vypíš tabuľku.
  • Načítaj n. Vytvor tabuľku nxn náhodných čísel z intervalu <100, 999>
    vypíš tabuľku
    utrieď riadky tabuľky podľa prvého stĺpca klesajúco a vypíš ju
    utrieď stĺpce tabuľky podľa prvého riadku klesajúco a vypíš ju
        (tu asi sort nepojde, odporúčam naprogramovať triedenie cez max)
  • Máme daný orientovaný graf s n vrcholmi a m hranami, ktorý zapíšeme do 2D vektora v2d tak, že v v2d[i] bude zapísaný vector kde bude výpis jeho priamych susedov.
    Načítaj start,ciel ( 0<= start,ciel <n ) a zisti, či existuje cesta zo start do ciel
        (odporúčam použiť ešte zásobník)
