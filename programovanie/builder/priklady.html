Príklady na pracu s Memo

  • Vypíšte do Memo čísla :
    1
    4
    9
    16
    ...
    2500
    Riešenie: 
    
	Memo1->Lines->Clear();	
	for (i=1;i<=50;i++) Memo1->Lines->Append(IntToStr(i));
  • Vypíšte do Memo1 prvých 15 členov Fibonnacciho postupnosti
    1, 1, 2, 3, 5, 8, ...
    každé číslo je tvorené pomocou súčtu dvoch predošlých
    Riešenie: 
    
	int a=1,b=1,c=2;
	Memo1->Lines->Clear();
	Memo1->Lines->Append(IntToStr(a));
	Memo1->Lines->Append(IntToStr(b));
	Memo1->Lines->Append(IntToStr(c));
	for (int i=4;i<=15;i++)
	{ a=b; b=c; Memo1->Lines->Append(IntToStr(c=a+b));
	}
  • Zapíšte do Memo1 50 náhodných dvojciferných čísel a do Edit1 zapíšte max. z týchto náhodných čísel

	for (i=1;i<50;i++) if (StrToInt(Memo1->Lines->Strings[0])> maxi) maxi = StrToInt(Memo1->Lines->Strings[i];
	Edit1->Text = IntToStr(maxi);

	Memo1->Lines->Clear();
	int a[100];	
	for (i=1;i<=50;i++) Memo1->Lines->Append(IntToStr(a[i]=10+random(90)));
	int maxi= a[0]);
	for (i=1;i<50;i++) if (a[i])> maxi) maxi = a[i];
	Edit1->Text = IntToStr(maxi);
