Stavba programu
Pre nás bude najdôležitejší ten program-súbor do ktorého budeme zapisovať svoje podprogramy. V C-čku sa volá Unit1.cpp. Premenné môžme vytvárať
vnútri fukcií - tieto budú lokálne (platné len v danej fukcii - funkciu main nevynímajúc)
mimo funkcií - tieto budú globálne - platné vo všetkých funkciách

Hlavný program - je ukrytý v projektovom súbore a zatiaľ tam nebudeme nič meniť.
Niečo tu meniť budeme až keď budeme vytvárať viacoknové aplikácie. Po stlačení Ctrl F12 a voľbe projekt sa nám ukážu nasledovné riadky:

WINAPI WinMain(HINSTANCE, HINSTANCE, LPSTR, int)
{
        try
        {
                 Application->Initialize();
                 Application->CreateForm(__classid(TForm1), &Form1);
                 Application->Run();
        }
        catch (Exception &exception)
        {
                 Application->ShowException(&exception);
        }
        catch (...)
        {
                 try
                 {
                         throw Exception("");
                 }
                 catch (Exception &exception)
                 {
                         Application->ShowException(&exception);
                 }
        }
        return 0;
}
