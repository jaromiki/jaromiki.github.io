Priateľské funkcie

Ak potrebujeme aby funkcia mala prístup ku privátnym údajom triedy môžme ju zadefinovať ako priateľskú funkciu. Priateľská funkcia teda nie je členskou funkciou ale napriek tomu má právo priameho prístupu ku privátnym objektom tiedy. Priateľská funkcia musí byť však deklarovaná (aspoň popis hlavičky) vo vnútri triedy s kľúčovým slovom frend. Jedna funkcia môže byť uvedená ako priateľská vo viacerých triedach.
Ukážka:
Unit1.cpp
// Deklarácia triedy
#include <iostream>
using namespace std;

class mojatrieda
{   int n, d;
    public:
	  mojatrieda(int nn, int dd)
	  {  n = nn; d = dd; }
	  friend bool rovnake(mojatrieda x);
};

//teraz zadefinujeme priatelsku fukciu
bool rovnake(mojatrieda x)
{ return x.n == x.d;}

int main()
{  mojatrieda x(1,2), y(5,5);
   cout << rovnake(x)?"v triede x su rovnake": "v triede x nie su rovnake" <<endl;
   cout << rovnake(y)?"v triede y su rovnake": "v triede y nie su rovnake" <<endl;
}
Poznámky:
* Priateľská fukcia nie je členskou funkciou a teda nemôže sa odvolávať na premenné n, d bez uvedenia mena inštancie triedy.
* Tak ako je zadeklarovaná mojatrieda je priateľská funkcia ako sa dostať ku privátnym dátam/objektom triedy.
* Nakoľko f. rovnake() nie je členskou funkciou nie je možný zápis x.rovnae - zahlási chybu
* Priateľská funkcia sa nededí! Použitie: Majme triedy
automobil
tir
kde v každej je zadefinovaný atribút max_rych. Ak potrebujeme tieto hodnoty porovnávať zadefinujeme v oboch triedach rovakú spriatelenú funkciu aby sme sa ku týmto údajom rýchlo dostali.
Unit1.cpp
// Predbežná deklarácia triedy tir
class tir;
class automobil
{ ...
 friend int porovnaj(automobil a, tir t);
}
class tir
{ ...
 friend int porovnaj(automobil a, tir t);
}
Predbe6ná deklarácia triedy tir musí byť použitá aby sme v triede mohli použiť identifikátor typu tir.
Je možné aby členská fukncia jednej triedy bola priateľskou funkciou inej triedy.
