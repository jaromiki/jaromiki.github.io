Príkazy

Príkazy tvoria reakciu na nejakú udalosť, ktorá nastala (najčastejšie na akciu užívateľa). Našou úlohou (úlohou programátora) je
  • stanoviť, čo sa má vykonať na nejakú akciu užívateľa (napr. stlačenie tlačídka)
  • preložiť túto akciu na postupnosť príkazov, ktoré Builder pozná
  • nájsť miesto kam tie príkazy napísať, aby sa v správny čas vykonali
Často chceme, aby sa niektorý príkaz za nejakých podmienok vykonal ale inokedy zas nevykonal.
Príklad zo života: Ak je chladnička prázdna -> ide sa nakupovať
V tomto príklade sa príkaz na nakupovanie nevykonáva vždy ale len ak nastane udalosť, že chladnička je prázna. V C-čku (všeobecne v každom programovacom jazyku) používame na tento cieľ vetvenie.

Neúplný if

Syntax - ako sa to píše
if ( podmienka) príkaz
podčiarknutý text je treba presne opísať (podmienka je vždy v zátvorkách)
  • podmienka - toto slovo nahradíme logickým výrazom (výraz, ktorého výsledná hodnota je TRUE alebo FALSE)
    (čo j jazyku C znamená, že ide o celočíselný výraz TRUE znamená rôzne od nuly)
    Napr. a>5
    Relačné (porovnávacie) operátory:
    • == ... test na rovnosť (napr. a == b ... ak je v premenných a,b rovnaká hodnota -> TRUE)
    • < ... menší (Napr. a< 0)
    • > ... väčší (Napr. a> 0)
    • <= ... menší alebo rovný (Napr. a<= 0)
    • >= ... väčší alebo rovný (Napr. a<= 0)
    • != ... nerovný (Napr. a !=5)
  • príkaz ... toto slovo nahradíme ľubovoľným príkazom jazyka C (napr. aj opäť vetvenie if)
Z kapitoly ako písať výrazy vieme že na konci jednoduchého príkazu je vždy bodkočiarka.
Sémantika - ako to funguje
Program keď má vykonať takýto príkaz tak:
  • vyhodnotí podmienku
  • ak dostane pravdu (!=0) vykoná príkaz uvedený za podmienkou
  • ak dostane nepravdu (==0) tak príkaz uvedený za podmienkou nevykoná
  • týmto je príkaz if ukončený
Poznámka:
Takýto if sa nazýva neúplný, lebo nie je určené čo máme robiť ak podmienka nepravdivá

Príkaz switch

Príkaz switch slúži tiež na vetvenie, ktorý príkaz sa má vykonať podľa nejakej podmienky, ale môžme si vyberať aj z viacerých príkazov ako z dvoch
Syntax - ako sa to píše
switch( výraz )
{
caseprípad1 : príkaz1 break; 
caseprípad2 : príkaz2 break; 
.............
default: príkaz ; ]
}
  • výraz - celočíselný alebo znakový výraz (prípady a výraz musia byť kompatibilného typu)
  • prípad - celočíselná alebo znaková konštanta
  • príkaz - ľubovoľný príkaz jazyka
  • vetva default je nepovinná

Nevýhody switch oproti if
... pri zadávaní prípadu nemožno používať premenné
... výraz za case nemôže byť textový (povolený je len char)
sémantika
  1. vyhodnotí sa výraz->podmienka
  2. vypočítaná hodnota sa hľadá v zadaných prípadoch, ak hodnotu nájde vykonáva všetky prípady=vetvy smerom nadol
    pokiaľ ho nevyhodí príkaz break ktorí ukončí celý príkaz switch
  3. ak hodnotu nenájde vykoná vetvu default ak je použitá (ak vetva default nie je neurobí nič)
Dôležité poznámky:
vetvu default dávajte až ako poslednú vetvu a tiež ju radšej ukončite príkazom break;
