Pole ako parameter

Niekedy potrebujeme preniesť pole do nášho podprogramu/funkcie a najlepšia cesta ako to urobiť je cez parametre. Keďže premenná typu pole je vlastne ukazovateľ tak pole sa prenáša referenciou (možno meniť jeho položky, čím sa budú meniť aj položky poľa v hlavnom programe. Tým, že sa vlastne do podprogramu prenesie len smerník na prvú položku poľa nemá podprogram ako vedieť koľko položiek pole vlastne má. Preto je zvykom pole prenášať dvomi položkami
  • adresa nultej položky poľa
  • počet prvkov poľa
Pole môžme zapísať ako parameter dvoma spôsobmi
  • typ prem[] ... napríklad: int a[], počet prvkov do zátvorky nezadávame
  • typ *prem ... je rovnocenné predošlému
Parameter pole môžme v podprograme používať pomocou indexov ale aj smerníkovou aritmetikou.
Ukážka

Prenos poľa do podprogramu
#include <iostream>
using namespace std;
int pole[10]={5,8,10,9,7,5,11,1,6,2} ;

int polemax(int *a,int n)    //pole je prenesené do a, počet je donesený do n
{ int mm=a[0];
  for (int i=1;i<n; i++) if (mm<a[i]) mm=a[i];
  return mm;
}
int main()
{  cout << "Maximálne číslo v poli:";
   for (int i=0;i<10; i++) cout <<pole[i] <<' ';
   cout << endl<<polemax(pole,10);
   cin.get();
}
Keby sme v podprograme polemax() menili položky poľa a[], tak by sa menili aj položky poľa pole[], lebo bývajú na tej istej adrese. Samozrejme, že ak by sme do premennej a zapísali inú adresu (pole a by bývalo na iných adresách), tak táto zmena by sa netýkala poľa pole[].
