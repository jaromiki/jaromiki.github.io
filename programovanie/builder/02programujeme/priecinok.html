Adresáre a súbory

Pre prácu s adresármi a súbormi máme k dispozícii viac rovnocenných nástrojov vo viacerých knižniciach, tiež možno používať príkazy z API. Ja budem popisovať príkazy z knižnice SysUtils.
Zistenie aktuálneho adresára
string GetCurrentDir()
vráti string v ktorom je celá cesta ku aktuálnemu adresáru. S každým procesom je spojený jeden adresár - tkz. aktuálny adresár. Všetky súborové operácie prebiehajú v ňom, pokiaľ nie je celou cestou (resp. relatívnou) určené inak.
Zmena aktuálneho adresára
bool SetCurrentDir(string cesta)
ak sa podarí zmeniť vráti True inak vráti False.
Zrušenie prázdneho adresára
bool RemoveDir(string Dir)
Zruší adresár uvedený ako parameter. Ak sa podarí vráti True inak False;
Vytvorenie prázdneho adresára
bool CreateDir(string Dir)
v aktuálnom adresári vytvorí adresár so zvoleným menom. Ak sa podarí vráti True, inak False.
Vytvorenie adresára aj s cestou
bool ForceDirectories(string s)
Zistenie, či súbor existuje
bool FileExists(string FileName)
ak súbor existuje vráti True
Premenovanie súboru
bool RenameFile(string OldName, string NewName)
zmení názov, ak sa nepodarí vráti False.
Zrušenie súboru
bool DeleteFile(string FileName)
zruší súbor s daným menom.
Kopírovanie súboru
bool CopyFile(string subor_zdroj, string subor_ciel, bool log_par)
funkcia API (v unite system)
log-par ... ak je pravda a subor-ciel existuje hlasi chybu a nekopíruje
log-par ... ak je nepravda a subor-ciel existuje prepíše tento súbor
ak sa nepodarí vráti 0
