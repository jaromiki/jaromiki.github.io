Druhé mocniny (štvorce)
Na súťažiach sa vyskytujú príklady: Nájdi všetky n aby V(n) bolo štvorec (sqrt(V(n) bolo celé). Nájdi všetky riešenia rovnice: V(n)=a^2 Na riešenie takýchto úloh používame špeciálne zbrane a je dobré ich poznať.
Spôsoby riešenia:
  1. Dokážeme, že V(n) nemôže byť štvorcom, lebo po delení číslom dáva "nesprávny" zvyšok
    1. V(n)%3=2 => V(n) nie je štvorec
    2. V(n)%4=2 alebo V(n)%4=3   => V(n) nie je štvorec
    3. V(n)%5=2 alebo V(n)%5=3   => V(n) nie je štvorec
    4. V(n)%a=b, kde b nie je štvorec   => V(n) nie je štvorec
  2. Dokážeme, že v prvočíselnom rozklade je aspoň jedno prvočíslo s nepárnou mocninou
  3. Dokážeme, že V(n) leží medzi dvoma po sebe idúcimi štvorcami b^2<V(n)<(b+1)^2
  4. Úpravou V(n) na tvar stvorec1-stvorec2 = konst, kde zo známej vzdialenosti štvorcov určíme o ktoré štvorce ide
  5. Úpravou rovnice - väčšinou na súčin
    (a^2-b^2 = k => (a-b)*(a+b) = k, a tu ak je k napr. prvočíslo je úloha jednoduchá)
Príklady:
  1. Nájdite n aby f(n)= n^2+n+2 bol štvorec
  2. Nájdite n aby f(n)=2005n^2 + 2002n + 2003 bol štvorec

  3. Nájdi všetky riešenia rovnice n^2 -5m= 27 v N*N
  4. Nájdite všetky n pre ktoré je 9^n+7 štvorec
  5. Nájdite n aby f(n)=n^4 + n + 7 bol štvorec
  6. Nájdite n aby f(n)=n^2 + n - 11 bol štvorec
  7. Nájdite n aby f(n)=n^2 + n + 1589 bol štvorec
  8. Dokážte, že 3n^2 -3n +2 nie je štvorec pre prirodzené n
  9. Určte všetky dvojice celých kladných čísel n a m, pre ktoré platí: 4^n + 4n^2 + 4 = m^2
  10. Dokážte, že n! nie je štvorec pre n>1
  11. Nájdite všetky n také, že 1!+2! + ... + n! je štvorec
  12. Nájdi 7 po sebe idúcich prirodzených čísel takých, že ich súčet je štvorec
  13. Dokáž, že súčet štvorcov za sebou idúcich prirodzených nie je štvorec
  14. Dokáž, že súčet 20-tich po sebe idúcich celých čísel nie je štvorec
  15. Dokáž, že rovnica a^2+b^2=3c^2 nemá v NxNxN riešenie
  16. Dokáž, že rovnica a^2=35b^2 nemá v NxN riešenie
  17. Nájdite všetky prirodzené čísla n také, že všetky tri čísla: 2n2+ 1,3n2+ 1 a 6n2+ 1 sú druhými mocninami celých čísel.
    Pomocou napr. excelu ľahko zistíme, že pre n <100 nemá riešenie, tak skúsime dokázať, že naozaj úloha nemá riešenie
    Sporom
    (3n^2+ 1)(6n^2+ 1) = 18n^4+ 9n^2+ 1 = 9n^2(2n^2+ 1) + 1 = (3n)^2(2n^2+ 1) + 1
    Čiže rozdiel štvorcov je 1 -> väčší štvorec musí byť 1, čo ľahko vylúčime a hotovo
  18. Dokážte, že neexistuje 20 po debe idúcich prirodzených čísel, takých že ich súčet je štvorcom
Podobne môžeme pracovať aj s "kubíkom"
Nájdi všetky prirodzené n aby 27^n + 37 bol kubík
Nájdite všetky celé čísla n, pre ktoré je číslo(n-4)^2-9 treťou mocninou prvočísla.
Určte všetky dvojice celých kladných čísel m,n, pre ktoré platí: 37 + 27^m=n^3.
