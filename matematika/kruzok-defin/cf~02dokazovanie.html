Dokazovanie
Na dokázanie nejakého tvrdenia máme 3 rôzne cesty. Niekedy je voľba správnej cesty cestu len voľbou aké ťažké to bude niekedy je voľba správnej cesty zásadná.
Dokázať tvrdenie možno: priamo, nepriamo, sporom alebo indukciou.
  • Dokazovanie - Priame Ak máme dokázať jednoduché tvrdenie (napr. dokážte, že sqrt(2)+sqrt(3)< sqrt(10)) postupujeme tak, že začíname zo všeobecne známeho tvrdenia o ktorom vieme že je pravdivé a povolenými úpravami ho upravíme do tvrdenia ktoré máme dokázať.
    Úloha: Dokáž, že sqrt(2)+sqrt(3)< sqrt(10)
    Riešenie:
    24<25 => (2*sqrt(2)*sqrt(3))^2>5^2 => 2*sqrt(2)*sqrt(3) >5 => 2+3+2*sqrt(2)*sqrt(3) > 10 => (sqrt(2)+sqrt(3))^2 > (sqrt(10))^2 => sqrt(2)+sqrt(3) > sqrt(10)
    Ako sme vedeli, 6e m8me začať práve s nerovnosťou 24<25? V skutočnosti upravujeme požadované tvrdenie pokiaľ nedostaneme pravdu a potom postupnosť tvrdení otočíme
    Najčastejšie bývajú matematické vety v tvare A=>B, kde A je predpoklad a B je tvrdenie, čo máme dokázať
    Odporúčaný postup teda je:
    Vyjdeme z tvdenia, ktoré je zaručene pravdivé, robíme povolené úpravy, využívame aj predpoklad až dostaneme tvdenie B
    Napr. Nech platí B1, úpravou dostaneme ak platí B1 tak platí B2, potom zas ak platí B2 potom platí B3 atd
    B1=>B2=>B3=>B4=>B5=>B ... čo bolo treba dokázať

    Príklad1:
    Nech x je prirodzené a ak x nie je delitelné 3 tak (x*x+2) je delitelné 3
    Dôkaz
    Využijeme predpoklad ... Z predpokladu je zrejmé že existuje prirodzené k také že x=3*k+1 alebo x=3*k+2
    1. Nech x=3*k+1: Teraz zistím či (x*x+2) je delitelné 3 ->po dosadení za x : ((3*k+1)*(3*k+1)+2)= 9k^2+6k+3=3*(3k^2+2k+1), čo je jasne delitelné 3
    2. Nech x=3*k+2: Teraz zistím či (x*x+2) je delitelné 3 ->po dosadení za x : ((3*k+2)*(3*k+2)+2)= 9k^2+12k+6=3*(3k^2+4k+2), čo je jasne delitelné 3

    Príklad2:
    Nech a,x sú prirodzené čísla, ak a delí x tak a delí x^2
    Dôkaz
    Využijeme predpoklad ... Z predpokladu je zrejmé že existuje prirodzené b také že x=a*b
    Teraz ukážeme, že a delí x^2: x^2 = (ab)^2 = a*(a*b*b), čiže a delí x^2
    Príklady:
    • pre n z N platí: 3 nedelí n => 3 delí n2+2
    • platí: sqrt(2)+sqrt(3)<sqrt(10)
    • Nech a,b sú prirodzené čísla, ak a delí b, tak a delí b2
    • Nech a,b,c >0, a > b potom (a+c)/(b+c)>1
    • Nech a,b > 0 potom (a+b)/2 >= sqrt(ab)
    • ku každému prirodzenému a existuje celé číslo b že platí: a^3-a = 6b
      (jednoduchšie povedané a^3-a je deliteľné 6)
    • Ak sa v štvoruholníku uhlopriečky rozpoľujú, potom štvoruholník je rovnobežník.
    • Druhá mocnina nepárneho čísla je tiež nepárne číslo.
    • Nech p je prvočíslo. p|a^2 => p|a
      Ukážte, že pre neprvočísla táto veta neplatí
  • Dokazovanie - Nepriame - A => B dokazujeme obmenu B´ => A´,
    princíp vysvetliť na vete "Ak x nedelí (a^2) tak x nedelí a", kde napísať priamy dôkaz je obtiažne
    príklady
    • Nech a je z N. Ak a^2 je deliteľné 3, potom aj a je deliteľné 3. Dokážte nepriamo.
    • a^2 je párne => a je párne
    • 5 delí (n^2 + 1) => 5 nedelí n
  • Dokazovanie - Sporom
    Použitie metódy dôkazu sporom môže byť:
    • Máme dokázať jednoduché tvrdenie A. Dôkaz je podobný nepriamemu dôkazu.
      Upravíme to na implikáciu (pravda => A). Z vlastnosti implikácie vieme, že takéto tvrdenie je rovnocenné pôvodnému.
      Teraz urobíme postup ako pri nepriamom dôkaze, čiže dokazujeme A´ => nepravda, čo znamená, že úpravou negácie A ,musíme dostať niečo, čo je zaručene nepravda.
    • Máme dokázať implikáciu A => B. Znegujeme tento výrok (A * B´) a "pokúsime" sa ho dokázať. Mali by sme prísť ku sporu.
      Najčastejšie sa zvykne dospieť k týmto sporom:
      a) A * B´ => A´, z predpokladu pravdivosti A ukážeme nepravdivosť A.
      b) A * B´ => B, z predpokladu nepravdivosti B ukážeme pravdivosť B.
      d) A * B´ => (not r), kde r je ľubovoľný známy pravdivý výrok.
    Príklady:
    • Dokáž, že sqrt(2) je iracionálne číslo.
    • Dokáž, že prvočísel je nekonečne veľa.
    • Dirichletov princíp: Nech k a n sú prirodzené čísla. Ak do n jamiek umiestnime aspoň kn+1 guľôčok, tak v aspoň jednej jamke bude aspoň k + 1 guľôčok.
    • Dirichletov princíp:Na večierku sa nachádza n ľudí, n > 1. Dokážte, že na večierku existujú dvaja ľudia s rovnakým počtom známych.
      Predpokladajte, že známosti sú obojstranné (ak ja poznám teba, tak aj ty poznáš mňa).
    • Nech a,b > 0 potom (a+b)/2 >= sqrt(ab), dokáž sporom
    • Nech m je celé. Ak je m^2 nepárne číslo, tak je aj m nepárne číslo.
      (dôkaz urobíme sporom: čiže znegujeme čo máme dokázať a veríme že nájdeme spor)
    • Dokáž, že rovnica a^2+b^2=3c^2 nemá v NxNxN riešenie
    • Dokáž, že rovnica a^2=35b^2 nemá v NxN riešenie
    • Dokáž, že rovnica a^2 -5b=27 nemá v NxN riešenie

Dokazovanie indukciou

Používame ak máme dokázať nejaké tvrdenie pre všetky prirodzené čísla (resp. pre jej nejakú nekonečnú podmnožinu)
Nech máme dokázať že platí: f(n) = g(n) pre všetky n. Dôkaz spočíva v dvoch krokoch
  1. Dokážeme, že výrok platí pre n=1, čiže f(1) = g(1)
  2. Predpokladajme, že platí f(n) = g(n) a dokážme že platí f(n+1) = g(n+1)

Úlohy
  • Dokáž že súčet prvých n prirodzených čísel je rovný n*(n+1)/2
    máme dokázať 1 + 2 + ... + n = n*(n+1)/2
    1. pre n = 1: Ľ = 1, P = 1*2/2 = 1, Ľ = P
    2. Nech platí: 1 + 2 + ... + n = n*(n+1)/2, treba dokázať 1 + 2 + ... + n + (n+1) = (n+1)*(n+2)/2
    1 + 2 + ... + n + (n+1)= n*(n+1)/2 + (n+1) = (n+1)*(n/2 +1) = (n+1)*(n+2)/2 = P
  • Dokáž že súčet prvých n neparnych prirodzených čísel je rovný n^2
  • Dokáž: 2^0 + 2^1 + 2^2 + ... 2^(n-1) = 2^n - 1
  • Dokáž: 6 | n^3-n, pre každe prirodzené n
  • Dokáž: 5 | n^5+4n, pre každe prirodzené n
  • Dokáž: 5 | 8^(2n)-3^(2n), pre každe prirodzené n
  • Dokáž: 7 | n^7-8n, pre každe prirodzené n
  • Dokáž: 16 | 9^(n+1)-8n-9, pre každe prirodzené n
  • Dokáž: Pre všetky prirodzené n>3 platí: n! > 2^n
  • Dokážte, že pre počet uhlopriečok Un v konvexnom n-uholníku platí vzorec: Un = n(n-3)/2
  • Dokážte, že súčet vnútorných uhlov vypuklého n-uholníka je (n-2)*pi
  • Dokážte, že súčet vnútorných uhlov ľubovoľného n-uholníka je (n-2)*pi.
  • Dokáž: -1 + 2 - 3 + 4 - 5 ...(-1)^n * n = ((-1)n(2n+1)-1)/4
  • Dokáž pre n >= 2 : 1/sqrt(1) + 1/sqrt(2) + ... + 1/sqrt(n) > sqrt(n)
  • Dokáž: 1/(n + 1) + 1/(n + 2) + 1/(n + 3) + ... 1/(n + n) > 13/24
  • Dokáž: 1/2 + 3/4 + 5/6 + ... (2n-1)/(2n) < 1/sqrt(2n+1)
