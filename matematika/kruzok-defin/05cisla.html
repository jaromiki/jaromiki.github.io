Rovnosť dvoch zlomkov - Veta

Nech zlomky a/b, c/d sú v základnom tvare a a/b=c/d => a=c a b=d
Táto veta nám pomôže riešiť veľa úloh
Príklad
Nech a,b sú nesúdeliteľné celé a platí ab/(a+b)=p/q, kde p,q sú prvočísla. Nájdi všetky a,b.
Ľahko sa ukáže že zlomok ab/(a+b) je v základnom tvare => ab=p, a+b=q =>a=1,b=p alebo b=1,a=p => [1,2],[2,1]

Úlohy
  • Vypočítaj NSD(x^2+1,x+1)

  • Nájdi nsd(2n+1, 2n-1)
  • Vypočítaj NSD(14m+17,21m+25)
  • Nájdi všetky n: n-3 | n^3-3
  • Nájdi n,m nesúdeliteľné (n+m)/(n^2-nm+m^2) = 8/73


  • Nájdi n,m nesúdeliteľné: n*m/(n+m)=3/4
  • Nájdi všetky n pre ktoré je zlomok (n^3-n^2-n+10)/(n-2) v základnom tvare
  • Nájdi všetky n,m pre ktoré platí: (n^3-n^2-n+10)/(n-2) = (m+1)/m
  • Nájdi všetky n,m pre ktoré platí: (n^3-n^2-n+10)/(n-2) = m
  • Nájdi všetky prvočísla p,q ku ktorým existuje prirodzené c:
    pq/(p+q) = (c^2+1)/c+1
  • Nájdi všetky prirodzené x,y: xy^2/(x+y)=p, p je prvočíslo




Niektoré zaujímavé rovnice
  • Nájdi všetky n pre ktoré je: n^5/5 + n^3/3 +7n/15 celé číslo
  • Nájdi riešenie v NxN rovnice: n^3 - 27^m = 37
  • V obore celých čísel vyriešte rovnicu MO2011/2012-škola-B1
    x^2 + y^2 + x + y = 4.
  • Predpokladajme, že pre kladné reálne čísla a, b, c, d platí
    ab + cd = ac + bd = 4 a ad + bc = 5.
    Nájdite najmenšiu možnú hodnotu súčtu a+b+c+d a zistite, ktoré vyhovujúce štvorice
    a, b, c, d ju dosahujú. MO2011/12-kraj-A4
    Pomoc: (a+b+c+d)^2=a^2+b^2+c^2+d^2+2ab+2ac+2ad+2bc+2bd+2cd=a^2+b^2+c^2+d^2+26>=2ad+2bc+26=36
    rovnosť nastane pri a=d a b=c, riešime sústavu 2ab=4, a^2+b^2=5 -> rieš [1,2], [2,1]
  • Nech a, b, c sú reálne čísla, ktorých súčet je 6. Dokážte, že aspoň jedno z čísel
    ab + bc, bc + ca, ca + ab nie je väčšie ako 8. MO-2010/11-doma-B3
  • V obore reálnych čísel riešte sústavu rovníc MO-2009/10-škola-B1
    sqrt(x - y2) = z - 1,
    sqrt(y - z2) = x - 1,
    sqrt(z - x2) = y - 1.
  • V obore reálnych čísel vyriešte sústavu rovníc
    x2 + 2yz = 6(y + z - 2),
    y2 + 2zx = 6(z + x - 2),
    z2 + 2xy = 6(x + y - 2).
    Pomoc: odčítať 1. rovnicu od 2. aj 3. rovnice a upraviť
    2003/2004-skol-A3
  • Nájdite všetky kvadratické trojčleny ax^2 + bx + c také, že ak ľubovoľný z koeficientov a, b, c zväčšíme o 1, dostaneme nový kvadratický trojčlen, ktorý bude mať dvojnásobný koreň.
    2003/2004-kraj-B2
  • Určte všetky polynómy P(x) také, že pre každé reálne číslo x je splnená rovnosť
    P(2x) = 8P(x) + (x - 2)2.
  • Nech P(x) = ax^2+bx+c je kvadratický trojčlen s nezápornými reálnymi koeficientmi.
    Dokážte, že pre ľubovoľné kladné číslo x platí
    P(x)*P(1/x) >= P2(1)
    Pomoc: Roznásobiť a upraviť.
  • Zistite, pre ktoré reálne čísla p má sústava rovníc
    x2 + 1 = (p + 1)x + py - z,
    y2 + 1 = (p + 1)y + pz - x,
    z2 + 1 = (p + 1)z + px - y
    s neznámymi x, y, z práve jedno riešenie v obore reálnych čísiel.
    Pomoc: Riešenia sú symetrické -> x=y=z
