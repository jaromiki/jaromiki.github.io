Vlastnosti čísel - deliteľnosť


Deliteľnosť je definovaná na množine celých čísel (v tejto časti uvažujem každé číslo ako celé)
Def.
Hovoríme, že a delí b (píšeme a|b ) práve vtedy keď existuje také c že a*c=b
(dá sa ukázať, že hľadané c je jediné)

Vlastnosti:
reflexívnosť: x|x
tranzitívnosť: x|y a y|z => x|z
x|y a x|z => x|(y+z)
x|y alebo x|z => x|(y*z)
x|y a x|(y+z) => x|z
x|y a x|z => pre ľubovoľné a,b platí: x|(ay+bz)

Ku ľubovoľnému číslu vytvorme množinu jeho deliteľov, potom pre túto množinu platí:
Lema: táto množina je konečná
Lema: táto množina obsahuje 1
Ak ku dvom číslam vytvoríme množinu deliteľov, tak tieto množiny majú neprázdny prienik

Def. najväčší spoločný deliteľ
Majmä dve celé čísla a,b. Vytvorme množinu M všetkých c takých že c|a a c|b
Maximum množiny M nazveme najväčší spoločný deliteľ (píšeme nsd(a,b))
Vlastnosti:
nsd(a,0) = |a|
nsd(a,a) = |a|
nsd(a,b) = nsd(b,a)
nsd(a,b) = nsd(|a|,|b|)
nsd(a,b) = nsd(a-b,b)
nsd(a,b,c) = nsd(nsd(a,b),c) = nsd(a,nsd(b,c))
Veta: nsd(a,b)=nsd(a%b,b)
Príklady:
Zisti koľko riešení v NxN má rovnica: NSD(a,b)+NSN(a,b) = 63
Príklad 2.1.6. Dokážte, že po sebe idúce hodnoty polynómu f(n) = n^4 + n^2 + 1 pre pridzené n sú súdeliteľné:
Riešenie:
f(1) = 3
f(2) = 21 = 3.7
f(3) = 91 = 7.13
f(4) = 273 = 3.7.13
f(5) = 651 = 3.7.31
Z prvých vypočítaných hodnôt sa zdá, že po sebe idúce čísla majú vždy spoločného deliteľa väčšieho ako 1, teda, že sú súdeliteľné. Ľahko sa môžeme presvedčiť o tom, že to tak bude skutočne pre ľubovoľné n. Platí totiž
f(n) = n^4 + n^2 + 1 = (n^2 - n + 1)(n^2 + n + 1),
f(n + 1) = [(n + 1)2 - (n + 1) + 1][(n + 1)2 + (n + 1) + 1] = (n^2 + n + 1)(n^2 + 3n + 3).
Nájdi nsd(f(n),f(n+1))
Nájdi nsd(n^3-1, n^2-1)
Nájdi nsd(2n+1, 2n-1)
Veta2: Nech a|b, a|c. Potom pre ľubovoľné celé x,y platí: a|bx+cy.    Urob dôkaz.
Veta3: Nech nsd(a,b)=d. Potom existujú také celé čísla x0,y0 že, d= ax0+by0.
Dôkaz: Nájdeme x1 a y1 také že dd=ax1+by1 bude najmenšie kladné ukážeme, že je rovné nsd
   nájdime q,r tak aby a=dd*q+r, kde 0<=r<dd
   teda r=a-dd*q=a-(ax1+by1)*q=a*(1-x1)+b*(-qy1)
   teda aj r sme zapísali v tvare lin. kombinácie ako aj dd a r je >=0 a menšie ako dd, čiže r musí byť 0
   čiže dd delí a. Podobne sa dá dokázať že dd delí aj b.
   podľa vety2 d|ax1+by1 a teda dd=d


Def. Súdeliteľnosť
Čísla a,b nazveme súdeliteľné <=> nsd(a,b)>1
Čísla a,b nazveme nesúdeliteľné <=> nsd(a,b)=1

Def. Kongruentnosť
Hovoríme, že čísla a,b sú kongruentné modulo m ak po delení m dávajú rovnaké zvyšky (a%m == b%m)
Iná def.
Hovoríme, že a je kongruentné s b modulo m a píšeme a = b (mod m), ak rozdiel a-b je deliteľný m.
Číslo m nazývame modul kongruencie.
Vlastnosti
(a+b)%m == (a%m + b%m)%m ... použitie chceme zistiť zvyšok po deleni súčtu viacerých veľmi veľkých čísel
(a*b)%m == (a%m * b%m)%m ... použitie chceme zistiť zvyšok po deleni súčinu viacerých veľmi veľkých čísel
(a^b)%m == (a%m)^b)%m ... použitie chceme zistiť zvyšok po deleni mocniny veľkých čísel
(a^(b*c))%m == (((a%m)^b)%m)^c)%m ... použitie chceme zistiť zvyšok po deleni mocniny veľkých čísel
Pre každé prvočíslo p a každé celé n platí :
    np = n ( mod p).
    inak povedané np %p = n % p
