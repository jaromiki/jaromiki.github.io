Príkazy
Výrazový príkaz
Pridaním bodkočiarky za výraz vytvoríme gramaticky korektný príkaz (5>4; je korektné)
a+=5;
Zložený príkaz
Ak tam, kde sa žiada 1 príkaz a my potrebujeme zadať príkazov viac uzavrieme ich zátvorkami. Túto konštrukciu voláme zložený príkaz.
{ a="Ahoj".fontcolor('blue'); document.write(a.fontsize("+1")); }
Vetvenie
neúplný if
syntax: if (log. výraz) príkaz1
ak je logický výraz TRUE prikaz1 sa vykoná inak sa pokračuje ďalším príkazom.
Pozn. - výraz môže byť aj celočíselný, ak je != 0, tak je TRUE
príklad: if ((m=prompt("Zadaj menovateľ","1")) ==0) alert("Chybny menovateľ!");
úplný if
syntax: if (log. výraz) prikaz1 else prikaz2
ak je logický výraz TRUE vykoná sa prikaz1 inak sa vykoná prikaz2.
Pozn. - výraz môže byť aj celočíselný, ak je != 0, tak je TRUE
príklad: if ((m=prompt("Zadaj menovateľ","1")) ==0) alert("Chybny menovateľ!") else document.write(n/m);
switch - prepínač
syntax:
switch (ordinálny výraz)
{
case konštanta1: príkaz1
case konštanta2: príkaz2
...
case konštantan: príkazn
default: príkaz        // nepovinné
}

sémantika:
Nájde prvú zhodu výrazu a niektorej konštanty a odtade vykonáva príkazy. Ak nenastane zhoda skočí na návestie default: - ak nie je nevykoná nič.
Na vyskočenie zo switchu sa používa príkaz break;
Príklad:

 cden=3;
 switch (cden)
 {
 1:den='pondelok';break;
 2:den='utorok';break;
 3:den='streda';break;
 4:den='štvrtok';break;
 5:den='piatok';break;
 default den='mimo rozsah';break;
 }
