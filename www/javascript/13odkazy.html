Hodnoty a odkazy

V Javascripte /ako aj v iných programovacích jazykoch) existujú 3 základné manipulácie s premennými
  • priradenie premennej do premennej
  • premenná ako parameter
  • porovnávanie dvoch premenných (hodnôt)
Každá takáto akcia sa môže vykonávať hodnotou alebo odkazom.
Hodnotou
Pri práci s hodnotou sú originál i hodnota dve nezávislé hodnoty. Používa sa pri primitívnych dátových typoch - číslo, logické hodnoty.
  • priradenie - vytvorí sa kópia originálu a zapíše sa na adresu premennej z pravej strany
  • parameter - zmena hodnoty parametra v podprograme nemá vplyv na skutočný parameter.
  • porovnávanie - ak dve premenné majú rovnaké hodnoty test na rovnosť vráti true.
Odkazom
Pri práci s odkazom sa presúvajú/porovnávajú len adresy a nie hodnoty. Používa sa pri štruktúrovaných dátových typoch - polia, objekty, funkcie.
  • priradenie - adresa hodnoty sa priradí do premennej vľavo
  • parameter - zmena hodnoty parametra v podprograme zmení aj skutočný parameter.
  • porovnávanie - ak dve premenné majú rovnaké hodnoty test na rovnosť nemusí vrátiť true.
Príklady 
	// *********************  Hodnota  *****************************
		var n = 1; 		// premenna n obsahuje hodnotu 1
		var m = n;		// kopirovanie hodnotou, v m je ina hodnota 1
		
		function pripocitaj(spolu, x)
		{ spolu += x; }
		
		// volanie funkcie
		pripocitaj (n,m);		// premenne n,m sa nezmenia, funkcia vlastne nerobi nic
		alert(n == m);			// vypíše true
		
	// *********************  Odkaz  *****************************
		var vianoce = new Date(2001, 11, 24);
		var slnovrat = vianoce; 		// obe premenne su na tej istej adrese
		
		slnovrat.setDate(21); 			// zmeni den v premennej slnovrat
		alert(vianoce.getDate();		// vypise hodnotu 21
		
		var pole = new Array[1,2,3];
		var x = 4;
		function pridaj(p,a)
		{ p[0] += x; p[1] += x; p[2] += x;}
		// volanie podprogramu
		pridaj(pole,x);
		alert(pole.join());				// vypise hodnoty 5,6,7
		
		var vianoce = new Date(2001, 11, 24);
		var slnovrat = new Date(2001, 11, 24); 		// obe premenne majú rovnakú hodnotu
		alert(vianoce == slnovrat)					// vypíše false
Reťazce
Otázka, čo sa deje pri priradení reťazca do reťazcovej premennej nie je dôležitá, lebo reťazce v JS sú nemenné. Nemáme žiadne nástroje, aby sme reťazec zmenili a teda či sa priradí do premennej adresa reťazca alebo hodnota je jedno.
Pri parametroch typu string tiež pri ich zmene v podprograme nenastane mimo podprogramu žiadna zmena.
Pri porovnávaní dvoch reťazcov sa porovnáva hodnota a nie adresa.
