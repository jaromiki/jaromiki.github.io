Document

Dokument predstavuje celú stránku, veľké množstvo vlastností preberá z tagu BODY. Najčastejšie sa stretávame s metódou write.
Metódy
open()
otvorí prúd do ktorého môžeme zapisovať obsah dokumentu - vyprázdni stránku
close()
uzatvorí prúd otvorený pomocou open(), az po uzavretí je dokument vidieť
write( text )
parameter text je zapísaný do dokumentu
  • ak je prúd otvorený pridá parameter-text na aktuálny koniec stránky
  • ak nie je prúd otvorený otvorí - a tým teda vyprázdni stránku a potom zapíše
writeln( text )
vkladá znak konca riadku (má význam ak sme v <PRE>
Vlastnosti
  • bgColor = " color " // farba pozadia
  • fgColor = " color " // farba písma
  • title = " text " // text z hlavičky stránky, textová vlastnosť určená len na čítanie
  • lastModifield ... reťazec s dátumom určený len na čítanie
  • aplets[] ... pole apletov použitých v stránke - umožní prístup ku ich verejným prem.
  • forms[] ... pole formulárov stránky (aj bez mena umožní prístup k formulárom stránky)
  • images[] ... pole všetkých obrázkov stránky, indexy idú od nuly
  • all[] ... pole všetkých objektov v poradí vzniku, umožní ku objektom pristupovať cez indexy
  • all.tags('nazov_tagu')[] ... pole všetkých objektov, ktoré sú uvedeného typu

Pomocou vlastnosti all, resp. all.tags môžme pomocou cyklu meniť vlastnosti veľkému počtu objektov v documente.
Pomocou document.all.tags('body')[0].width (height) môžme čítať i meniť size okna

Window

V tomto objekte sú vlastnosti okna, v ktorom je dokument vypísaný.
Metódy
  • alert( text ) // vypíše oznam v dialógovom okne
  • okno.blur () // pošle okno do pozadia
  • focus() // pošle okno dopopredia
  • okno.clearInterval(intervalid) // procedura, zruší periodické vykonávanie príkazov - vypne daný časovač
  • okno.setInterval(kod,interval) // funkcia, zapne časovač - nastaví opakovanie kódu každých interval ms, vrati hodnotu intervalid
  • okno.setTimeout(kod, cas) // pozastaví vykonávanie na zadaný čas (ako sleep, delay)
  • moveto() // presun okna na dané súradnice obrazovky
  • okno.print() //vytlačí dokument
  • resizeBy(dsir,dvys) // zmení veľkosť okna o danú veľkosť
  • resizeTo(sir,vys) // zmení veľkosť okna na danú veľkosť
  • okno.scrollBy(dx,dy) // roluje okno o zadaný počet bodov
  • okno.open(url,nazov,prvky) // funkcia vráti názov okna použitelný v odkazoch target (zvonka)
    • url - nepovinné, adresa stránky, ak prázdny reťazec je stránka prázdna
    • nazov - meno okna na prístup ku vlastnostiam, metódam okna zvnútra okna
    • prvky - ktoré vizuálne prvky má okno mať, ak neuvedieme bude mať všetky, ak jeden uvedieme ostatné nemá
      • fullscreen=yes|no ... na celú obrazovku?
      • height= cislo ... určí výšku okna
      • width= cislo ... určí šírku okna
      • left= cislo ... x-sová súradnica okna na obrazovke
      • top= cislo ... y-nová súradnica okna na obrazovke
      • location=yes|no ... vstupný riadok áno/nie
      • menubar=yes|no ... menu áno/nie
      • toolbar=yes|no ... paleta nástrojov áno/nie
      • scroolbar=yes|no ... zobrazenie posuvníka
      • status=yes|no ... stavový riadok áno/nie
      • resizable=yes|no ... či budú nástroje na zmenu veľkosti
Vlastnosti
  • frames[] ... pole rámcov, frames.length určuje koľko ich je. Typ položky je window.
  • screen ... objekt kde je popis obrazovky na ktorej je okno
  • status ... reťazec v stavovom riadku
  • event ... objekt na popis poslednej udalosti
    clientX, clientY ... súradnice miesta, kde sme klikli (pohli) myšou
    Udalosti
    onblur ... pri odchode do pozadia
    onerror ... pri výskyte chyby
    onload ... pri úplnom načítaní (document.close())
    onunload ... ak opúšťame aktuálny dokument
    Ukážka
    nove=window.open("","okno","status=no"); //vytvorí nové okno bez statusu
    nove.resizeTo(165,30); //určí oknu vaľkosť
    nove.document.open(); // open otvorí prúd do okna (vymaže ho zároveň)
    nove.document.write("<html>"); // prvý text do okna
    nove.document.write("<head>\n");
    nove.document.write("<META HTTP-EQUIV='Content-Type' CONTENT='text/html; charset=windows-1250'>\n");
    nove.document.write('<title>Pokus\n');
    nove.document.write("</head>\n"); // koniec hlavičky
    nove.document.write("<body>\n"); // a začína telo
    nove.document.write("<div >\n"); // vytvorím objekt v tele dokumentu
    nove.document.write(" Text"); // tento text sa zobrazí
    nove.document.write("</div>\n"); // koniec objektu div
    nove.document.write("</body>"); // koniec tela
    nove.document.write("</html>"); // koniec html
    nove.document.close(); // a uz by okno aj malo vidieť
