Tu vidieť hlavný zmysel PHP - vkladať do HTML stránky-kódu texty, ktoré môžu obsahovať aj HTML príkazy.
prikaz echo, print
Kedže hlavnou prácou PHP kódu je vkladanie textu a HTML príkazov do HTML kódu hlavný je príkaz na výstup textu Používame príkaz echo alebo synonymum print. Syntax je jednoduchá - za medzeru uvedieme textový výraz (ak uvedieme číselný výraz, bude prevedený na textový). Výsledok výrazu bude vložený do HTML kódu na to miesto kde sa nachádza PHP kód.
Konštanty
Konštanty môžu byť textové, celočíselné, číselné-desatinné, logické
  • textové - ľubovoľný text uzavretý v
    • úvodzovkách - "toto je textová konštanta"
    • apostrofoch - 'aj toto je textová konštanta'
    Vkladanie špeciálnych znakov do textu:
    \n - LF (enter)
    \r - CR (vo Win je Enter tvorený dvojicou znakov CR LF)
    \\ - \ opačné lomítko
    \$ - $
    \' - '
    \" - "
    Rozbaľovanie premenných v textu:
    Text može obsahovať premenné (znak $ a postupnosť písmen, číslic a podtržníka), tieto sú v textovej konštante " ... " nahradené obsahom premennej.
    KódVýsledok
    <body>
    <?
    $a='Ahojte všetci'; // do premennej $a sa zapise text Ahojte všetci
    echo "Premenná \$a : $a";
    // prvy vyskyt $a sa vypise ako $a - rozbalenie prekazilo spätné lomitko     		Premenná $a : Ahojte všetci
  • Celočíselné - celé čísla môžme zadávať
    • desiatkové: 5, +12, -18000
    • osmičkové: 055 (=5*8+5=48), 078 (chyba, číslo začínajúce 0 nemôže obsahovať cifru>7)
    • šestnástkové (hexa): 0x12, 0xcc
  • desatinné - používame desatinnú bodku
    1.3, -520.15, 1.2e6, -5.55e-4, 1e3, 3e-3
  • logické
    true - pravda (1)
    false - nepravda (0)
