Pole

Pole používame, ak chceme do jednej premennej zapísať viacero hodnôt. Prístup ku hodnotám je pomocou indexu. Narozdiel od pascalu (C-čka) tu môže byť každá položka poľa iného typu a premennú typu pole nie je treba deklarovať.
Vytvorenie poľa
  • Pole vznikne ak použijeme položku poľa napr. $pole[5]=10; // vzniklo pole $pole s jednou položkou
  • Vytvorenie pomocou konštrukcie array
    $p=Array(10, 15, 'Ahoj'); //vytvorí pole $p s položkami $p[0], $p[1], $p[2] s vymenovanými hodnotami
    štandartne je číslovanie poľa od 0
  • číslovanie od inej hodnoty
    $p=Array(11 => 10, 15, 'Ahoj'); //vytvorí pole $p s položkami $p[11], $p[12], $p[13] s vymenovanými hodnotami
  • Vztvorenie poľa pomocou funkcie range(min, max)
    Vytvorí pole s hodnotami min až max
  • Vytvorenie asociatívneho poľa
    pole, kde indexy nie sú číselné (ordinálne) alebo ak aj sú číselné nie sú súvislé
    $x['meno']='Ján'; $x['priezvisko']='Pekný';
    $z=Array('meno' => 'Ján', 'priezvisko => 'Pekný');
    //oba tieto postupy vytvoria dvojprvkové pole, položky majú meno a hodnotu.
Prechod poľa
Na prechod cez všetky položky poľa používame cyklus. Najjednoduhšie je prejsť poľom, kde indexy sú číselné a súvislé
$p=Array(10, 15, 'Ahoj');
for ($i=0; $i<3; $i++) echo "index: $i, hodnota: ".$p[$i]."
";
//počet prvkov poľa môžme určiť pomocou funkcie count(pole).
for ($i=0; $i<100; $i++) $b[$i]=$i*$i; //naplní 100 prvkové pole hodnotami druhých mocnín
Prechod asociatívneho poľa
Uvedený postup možno použiť aj na očíslované polia.
Máme dve možnosti použiť cyklus foreach - vytvorený špeciálne na prechod poľa alebo pomocou funkcií reset(), count(),...
  • foreach
    $x['meno']='Ján'; $x['priezvisko']='Pekný';
    foreach ($x as $akt) echo "$akt
    "; // vypíše hodnoty poľa $x
    foreach ($x as $i=>$akt) echo "index:$i, hodnota: $akt
    "; // vypíše indexy a hodnoty poľa $x
  • prechod pomocou ukazovateľov
    • count(pole) ... vráti počet položiek poľa
    • reset(pole) ... nastaví ukazovateľ na začiatok poľa
    • current(pole) ... vráti aktuálnu hodnotu poľa, ak nie je vráti false
    • key(pole) ... vráti aktuálnu hodnotu indexu
    • next(pole) ... presunie ukazovateľ na nasledovný prvok

    reset($x);
    while (current($x))
    { ľub. príkazy;
    next($x); }
    // tento príklad prejde celým poľom, ak je prázdne neurobí nič

Funkcie na triedenie poľa
  • sort(pole) ... výsledok je utriedené pole podľa hodnoty, indexy sú od 0
  • asort(pole) ... výsledok je utriedené pole podľa hodnoty, indexy sú zachované
  • ksort(pole) ... výsledok je utriedené pole podľa indexov
Testy na premenné
Často potrebujeme zistiť, či premenná existuje, aký má typ, či je prázdna
  • isset(prem) ... či existuje
  • isempty(prem) ... či je prázdna (prázdny reťazec, pole, hodnota null)
  • is_bool(prem),is_text(prem), is_int(prem), is_array(prem);
